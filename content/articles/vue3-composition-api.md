# Vue 3 Composition API 入门指南

Composition API是Vue 3的核心特性之一。

## 什么是Composition API

Composition API 是一种新的代码组织方式，允许你更好地组织和重用代码逻辑。

### 优势

1. 更好的逻辑复用
2. 更灵活的类型推断
3. 更清晰的代码结构

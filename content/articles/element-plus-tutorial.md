# Element Plus 组件库使用教程

Element Plus是一个基于Vue 3的组件库，提供了丰富的UI组件。

## 安装

```bash
npm install element-plus --save
```

## 基本使用

```vue
<template>
  <el-button type="primary">主要按钮</el-button>
</template>
```

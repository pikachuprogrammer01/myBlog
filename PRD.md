# PRD — myBlog 产品需求文档

## 1. 产品概述

### 1.1 产品定位
myBlog 是一个**全栈个人技术博客系统**，面向开发者群体，提供技术文章阅读、面试题练习、工具推荐等功能。系统同时作为**毕业设计项目**，展示全栈开发能力。

### 1.2 目标用户

| 用户角色 | 描述 | 核心需求 |
|----------|------|----------|
| 读者（游客） | 浏览技术文章的普通访客 | 阅读文章、搜索内容、浏览面试题 |
| 注册用户 | 已注册并登录的用户 | 评论互动、点赞收藏、个人中心 |
| 管理员 | 博客运营者 | 内容管理、数据统计、用户管理 |

### 1.3 核心价值
- 为开发者提供高质量中文技术文章（Java、Vue、前端工程化等方向）
- 提供面试题库，支持分类浏览和难度筛选
- 提供开发者常用工具推荐
- 完整的用户互动体系（评论、点赞、收藏）

## 2. 功能需求

### 2.1 博客文章系统（P0）

#### 2.1.1 文章列表
- 首页展示文章卡片列表，支持网格/列表两种布局切换
- 每页显示 6 篇文章，底部分页导航
- 文章卡片展示：封面图、标题、摘要、发布日期、阅读量、标签
- 首页顶部轮播图展示精选文章（3 篇有封面图的文章）

#### 2.1.2 文章详情
- Markdown 渲染正文，支持代码高亮、表格、脚注、任务列表
- 自动生成文章目录（TOC），浮动侧边导航
- 阅读进度条（底部细条）
- 文章元信息：分类、标签、阅读量、发布日期
- 上一篇/下一篇导航
- 分享、打印、导出 Markdown 功能

#### 2.1.3 文章分类与标签
- 按分类浏览文章，展示各分类文章数量
- 标签云页面，按热门程度着色和排序
- 支持按标签筛选文章列表

#### 2.1.4 文章搜索
- 全文关键词搜索，结果高亮展示
- URL 参数同步（`?q=关键词`），支持直接分享搜索结果
- 防抖搜索建议，热门搜索标签快捷入口

#### 2.1.5 文章归档
- 按年份/月份时间线展示所有文章
- 统计面板：文章总数、覆盖年份、标签数量、最活跃年份
- 快速年份导航

### 2.2 用户系统（P0）

#### 2.2.1 注册
- 用户名 + 密码注册，用户名支持字母数字下划线
- 密码强度实时检测（弱/中/强三级）
- 确认密码一致性校验
- 注册成功自动登录

#### 2.2.2 登录
- 用户名 + 密码登录
- JWT 令牌认证，7 天有效期
- 登录状态持久化（localStorage），刷新不丢失
- 登录/注册页全屏布局（无页头页脚）

#### 2.2.3 个人中心
- 头像上传（Canvas 前端裁剪为 300×300）
- 用户统计：评论数量、注册天数
- 我的评论列表（含所属文章链接）
- 退出登录

#### 2.2.4 我的收藏
- 已收藏文章列表，卡片形式展示
- 每篇显示封面图、标题、摘要、收藏日期
- 支持取消收藏

### 2.3 评论系统（P0）

#### 2.3.1 评论功能
- 登录后可发表评论，支持 Markdown 格式
- 评论工具栏：加粗、斜体、链接、代码插入
- 评论预览切换
- 嵌套回复（楼中楼），支持多层级

#### 2.3.2 评论排序
- 最新优先 / 最早优先 / 最热优先三种排序
- 最热按点赞数降序

#### 2.3.3 评论互动
- 评论点赞/取消点赞
- 评论者本人和管理员可编辑/删除
- 软删除（前端不可见，数据库保留）

### 2.4 点赞与收藏（P1）

#### 2.4.1 文章点赞
- 文章详情页点赞按钮，登录后可操作
- 点赞状态实时展示
- 点赞数统计

#### 2.4.2 文章收藏
- 文章收藏/取消收藏
- 收藏状态图标切换
- 收藏列表查看

### 2.5 面试题库（P1）

#### 2.5.1 题库分类
- 六大分类：JavaScript、Vue.js、CSS、算法、工程化、项目
- 每个分类展示题目总数及难度分布（简单/中等/困难）

#### 2.5.2 题目浏览
- 按分类浏览题目列表
- 难度筛选（全部/简单/中等/困难）
- 分页展示，10 条/页

#### 2.5.3 题目详情
- Markdown 渲染题目内容（含代码块、表格等）
- 自动生成目录
- 难度标签展示
- 相关题目推荐
- 评论区（与博客评论共用评论系统）

### 2.6 工具推荐（P1）
- 开发者工具分类展示（Vue 生态、前端工具、Java 生态、设计资源等）
- 工具卡片：名称、描述、图标、跳转链接
- 分类筛选

### 2.7 联系表单（P2）
- 登录后可提交联系信息
- 邮件自动通知（Nodemailer 发送）
- 频率限制：每用户每天 10 次

### 2.8 管理后台（P0 — 管理员专属）

#### 2.8.1 数据概览
- 统计卡片：文章数、评论数、用户数、总浏览量
- ECharts 饼图：文章分类分布

#### 2.8.2 文章管理
- 文章列表（含浏览量、点赞数、评论数、收藏数）
- 新建/编辑文章：标题、slug、分类、标签、封面图、Markdown 正文
- Markdown 文件上传解析
- 发布/草稿状态切换
- 批量操作
- 8 条/页分页

#### 2.8.3 评论管理
- 所有评论列表，支持搜索和排序
- 置顶/取消置顶
- 软删除/永久删除
- 批量删除
- 8 条/页分页

#### 2.8.4 标签管理
- 标签 CRUD
- 文章数量关联展示
- 8 条/页分页

#### 2.8.5 工具管理
- 推荐工具 CRUD：名称、URL、描述、分类、图标、排序
- 8 条/页分页

#### 2.8.6 面试题管理
- 题目 CRUD：标题、分类、难度、标签、摘要、Markdown 内容
- 8 条/页分页

#### 2.8.7 用户管理
- 用户列表，角色编辑（admin/user）
- 删除用户
- 重置密码
- 8 条/页分页

#### 2.8.8 数据操作
- 各面板数据导出为 Excel（全量数据，不受分页限制）
- 清空所有评论
- 重置系统数据
- 测试邮件配置

### 2.9 关于页（P2）
- 作者个人介绍
- 技能展示（进度条形式）：Vue、Java、MySQL、Spring Boot 等
- 成长时间线
- GitHub / 邮箱联系入口

## 3. 非功能需求

### 3.1 性能
- 首页首屏加载 ≤ 3 秒（3G 网络）
- API 响应时间 ≤ 500ms（P95）
- 文章列表缓存策略（localStorage 缓存 → 静态 JSON → API 刷新）

### 3.2 兼容性
- 支持 Chrome、Firefox、Edge 最新两个大版本
- 移动端响应式适配（断点：768px、1024px）
- 暗色模式自动跟随系统设置（`prefers-color-scheme`）

### 3.3 安全
- 密码 bcryptjs 加盐哈希存储（10 轮）
- JWT 令牌认证，7 天自动过期
- CORS 白名单限制
- 评论编辑/删除权限校验（本人或管理员）
- 管理后台路由守卫（前端）+ 接口鉴权（后端）
- 数据库 TLS 加密连接

### 3.4 可维护性
- 组件化开发，Vue 3 Composition API
- Pinia 状态管理，API 服务层分离
- ESLint + Prettier 代码规范
- 单元测试覆盖核心逻辑

### 3.5 SEO
- 语义化 HTML 标签
- sitemap.xml
- 合理的 title/meta 标签

## 4. 用户流程

### 4.1 首次访问流程
```
访问首页 → 浏览文章列表 → 点击文章 → 阅读全文 → 浏览分类/标签 → 搜索文章
```

### 4.2 用户注册流程
```
点击登录 → 切换到注册 → 填写用户名密码 → 注册成功自动登录 → 跳转首页
```

### 4.3 互动操作流程
```
阅读文章 → 点击点赞/收藏 → 提示登录 → 登录 → 返回文章 → 完成操作
阅读文章 → 写评论 → 提示登录 → 登录 → 返回文章 → 发表评论
```

### 4.4 管理员流程
```
登录管理员账号 → 导航栏用户菜单 → 管理后台 → 选择管理Tab → 执行操作
```

## 5. 数据指标

### 5.1 核心指标
- 文章总数 ≥ 20 篇
- 分类覆盖 ≥ 5 个技术方向
- 系统正常运行时间 ≥ 99%

### 5.2 用户指标
- 页面浏览量（API 记录）
- 评论互动量
- 用户注册数
